# Workshop 03 — Skills Book
## เวิร์กช็อป 03 — เขียนหนังสือ Skills

เรียนเรื่อง Oracle skills ecosystem — ทั้ง skill creation, book writing pipeline, และ Thai prose quality

### สิ่งที่เรียนรู้

- `/oracle-write-complete-book` pipeline: outline → parallel draft → compile → PDF
- kien-thai: เขียนภาษาไทยให้เป็นธรรมชาติ ไม่ใช่ AI-generic
- typst: Thai font rendering (Sarabun) + code blocks (Fira Code)
- Parallel Sonnet agents สำหรับ draft แต่ละบทพร้อมกัน

> "เขียนจาก source material จริง ไม่ใช่จาก memory — ข้อมูลผิดได้"
