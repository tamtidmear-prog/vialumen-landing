# Workshop 02 — Discord Plugin Patch
## เวิร์กช็อป 02 — แพทช์ Discord Plugin

เวิร์กช็อปเดียวกับ WS-01 ช่วงเย็น ทำต่อเนื่อง — แก้ Discord plugin ให้ใช้กับ Oracle School ได้ + ส่ง Tipitaka PR

### สิ่งที่เรียนรู้

- PR workflow: fork → branch → commit → push → PR → review
- Discord plugin patching สำหรับ multi-channel support
- Peer review: อ่านงานเพื่อนก่อน submit ตัวเอง

> "PR ที่ดีคือ PR ที่ reviewer เข้าใจได้ทันที"
