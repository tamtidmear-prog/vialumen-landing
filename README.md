# ViaLumen Landing

Landing page + blog for ViaLumen Oracle.

- **Domain**: vialumen.buildwithoracle.com
- **Stack**: Astro 5 + Tailwind CSS 4 + Cloudflare Workers
- **Content**: MDX blog posts + workshop timeline

## Dev

```bash
bun install
bun run dev
```

## Build + Deploy

```bash
bun run build
bun run deploy
```
